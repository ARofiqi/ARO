Siapa Saya?

Kenalkan kembali, saya Ahmad Rofiqi. Saya adalah Web Developer yang mengawali langkah dari menulis kode HTML/CSS di layar kecil *smartphone*, dan kini berfokus pada perancangan arsitektur *frontend* modern dan pengembangan aplikasi berbasis React & Next.js.

Saya memiliki ketertarikan mendalam terhadap bagaimana sebuah aplikasi web dapat bekerja selayaknya aplikasi *native* di *smartphone* pengguna. Melalui eksplorasi dan pengerjaan proyek akhir mengenai Strategi Offline-First pada sistem POS, saya berkomitmen menghadirkan produk digital yang tangguh tanpa batas koneksi.
